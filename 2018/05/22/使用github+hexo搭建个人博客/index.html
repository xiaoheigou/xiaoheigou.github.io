Quick Start

第一步：看官方文档

官方文档写的很详细 ：https://hexo.io/docs/index.html
参考资料：https://zirho.github.io/2016/06/04/hexo/
主要是安装prerequisite and hexo

第二步：项目初始化

  • 在任意文件夹运行hexo init 命令，会发现文件夹下生成了很多东西
  • 修改hexo init 之后的_congih.yml 文件，如下

第三步：部署

  • hexo g 根据source中的blog.md，生成html文件到public目录下
  • hexo d 将public目录push到github仓库下的repository中(仓库地址在第二步中配置过了)

第四步：选择一个合适的主题

  • git clone 一个主题到 themes 文件夹下，并在_config.yaml中添加 theme: aath
  • 返回第三步，主要是更新blog，重新部署

第五步：创建文章

  • hexo new “title” 创建一个blog，用markdown语法写些文章
  • hexo d -g 重新部署

注意点：

  • config.yaml 有两个，一个是全局配置 ，主要配置github地址和一些通用配置，还有一个是thems文件夹下的主题配置，主要是配置些主题的内容
  • 关于页面和分类页面需要在source文件夹下面手动创建，先创建一个about文件夹，然后在里面创建index.md.不同主题可能方法不同。

可能的坑：

安装渲染插件时候可能失败

渲染器安装失败：https://github.com/tufu9441/maupassant-hexo/issues/272
